# **Cloth Donation Platform**

## **Purpose**
The Cloth Donation Platform is designed to simplify the donation process by connecting donors and needy individuals. Users can easily explore active donation campaigns, contribute items, and track the progress of their contributions. This platform aims to promote community support and make a positive impact on society.

## **Live URL**
[View Live Project](https://cloth-donation-5cd35.web.app)

## **Key Features**
1. **Donation Campaigns Page**:
   - Displays all active donation campaigns with cards including images, titles, descriptions, and divisions.
   - A “Donate Now” button for navigation to campaign details.

2. **Donation Details Page**:
   - Displays detailed information about a selected campaign.
   - Includes a donation form for users to provide item details, pickup location, and additional notes.
   - Shows a toast message upon form submission.

3. **Authentication**:
   - Login and Registration pages for user authentication.
   - Supports Google Login.
   - Protects private routes such as Dashboard and Donation Details.

4. **Dashboard**:
   - Displays user profile information (image, email, name).
   - Allows users to update their profile information.

5. **Forgot Password**:
   - Functional feature to reset the user’s password via email.

6. **How to Help**:
   - Provides instructions for donation, collection points, and supported divisions.

7. **Dynamic Animations**:
   - Smooth animations powered by AOS (Animate On Scroll).

8. **Responsive Design**:
   - Tailwind CSS ensures the platform is fully responsive and user-friendly on all devices.

## **NPM Packages Used**
- **React**: Core library for building the UI.
- **React Router DOM**: For routing and navigation.
- **Firebase**: For authentication and backend services.
- **Tailwind CSS**: For styling the UI.
- **AOS (Animate On Scroll)**: For scroll-based animations.
- **React Toastify**: To show toast notifications.
